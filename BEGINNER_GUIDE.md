# 🏁 电脑小白超详细上手指南

如果你对代码、命令行不太熟悉，请按照以下步骤一步步操作。我们将带你从零开始，在你的电脑上把这个平台跑起来。

> 💡 **提示**：如果你对“环境”、“依赖”、“Python”等词汇感到困惑，建议先阅读 **[GLOSSARY.md (术语通俗解释)](GLOSSARY.md)**，我们用建筑学和城乡规划的视角为你做了类比。

---

## 第一步：准备运行环境 (只需做一次)

### 1. 下载并安装 Python
Python 是运行这个平台的基础软件。
- **下载地址**：[Python 3.12.3 官网下载](https://www.python.org/ftp/python/3.12.3/python-3.12.3-amd64.exe) (点击直接下载 Windows 版)。
- **安装重点**：双击安装包后，**务必勾选底部的 "Add Python to PATH"**（这步非常重要，否则后面会报错）。
- 点击 "Install Now" 直到完成。

### 2. 下载本项目代码
- 在本页面顶部找到绿色的 **"Code"** 按钮。
- 点击 **"Download ZIP"**。
- 下载完成后，找一个你喜欢的文件夹（比如桌面或 D 盘）解压它。

---

## 第二步：快速配置与启动

现在，进入你刚才解压出来的文件夹 `ultimate-design-main`。

### 1. 打开命令行窗口
- 在文件夹顶部的地址栏（显示路径的地方）直接输入 `cmd` 然后回车。
- 此时会弹出一个黑色的窗口，光标就在你当前的文件夹路径下。

### 2. 自动化一键配置 (推荐)
为了方便小白，我们在根目录准备了 `setup_env.bat`。
- 在黑色窗口输入：`setup_env.bat`
- 回车运行。它会自动帮你创建虚拟环境并安装所有需要的依赖。
- 如果提示“安装完成”，说明环境已经配好了。

### 3. 启动平台
- 以后每次想运行，只需在文件夹地址栏输入 `cmd`，然后在黑窗口输入：
  ```powershell
  streamlit run app.py
  ```
- 稍等几秒，你的浏览器会自动打开一个网页，这就是我们的决策支持平台！

## 第三步：常见问题救急

### Q: 提示 'python' 不是内部或外部命令
**原因**：安装 Python 时没有勾选 "Add Python to PATH"。
**解决**：卸载 Python 重新安装，记得勾选那个复选框；或者联系懂电脑的朋友手动配置环境变量。

### Q: 安装依赖太慢了
**原因**：默认下载源在国外。
**解决**：你可以尝试手动输入这个加速命令：
```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 页面上显示“AI 引擎未就绪”
**解释**：这**不是错误**。因为生图 (Stable Diffusion) 和对话 (Ollama) 需要极高性能的显卡和额外的软件配置。
**结论**：即使显示未就绪，你依然可以查看已有的 3D 地图、地块数据和成果展示，不影响基础功能的使用。
