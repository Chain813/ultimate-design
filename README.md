# UltimateDESIGN

**长春伪满皇宫周边街区微更新与城市设计决策支持平台**

UltimateDESIGN 是一个面向城乡规划专业城市设计课程、毕业设计和研究展示的 Streamlit 应用。项目严格按 **13 个专业阶段** 组织页面，贯穿了「前期现状分析 -> 中期概念策略 -> 后期设计深化与导出」的完整循证规划工作流。

## 🌟 核心特色与功能

1. **13 阶段独立工作流**：将城市设计拆解为 13 个标准化步骤，每个步骤包含独立功能面板、AI 图纸提示词生成与阶段答辩总结。
2. **41 种专业图纸模板覆盖**：内置 41 种基于 `Stable Diffusion` 和 `ControlNet` 的提示词模板，自动提取空间数据（如建筑高度、POI 活力）注入提示词。
3. **13 个阶段动态统计图**：利用 Plotly 在每个阶段的答辩小结自动生成对应的分析图表（如：MPI 更新潜力排行、多主体共识度雷达、分期实施甘特图等）。
4. **LLM 循证推演与答辩总结**：深度挂载本地 **Gemma 4**，通过跨阶段数据总线 (`stage_data_bus`)，一键生成去 AI 味、专业术语饱满的学术段落。
5. **高性能与局部刷新**：采用数据格式缓存优化（`@st.cache_data`）、JS 异步地图挂载与 Streamlit 碎片化重绘（`@st.fragment`），保障 150 公顷尺度的极速渲染。

## 🧩 核心底层模块 (src/)

- `workflow/stage_data_bus.py`: **跨阶段数据总线**。管理全链条证据传递，保障后一阶段可调用前一阶段的统计与诊断结论。
- `ui/module_summary.py`: **阶段答辩小结组件**。为 13 个页面注入底部的学术总结面板、动态 Plotly 图表与 Gemma 一键推演按钮。
- `ui/app_shell.py`: **全局外壳**。负责全宽沉浸式 Top Nav 导航映射。
- `engines/drawing_prompt_templates.py`: **提示词模板中心**。注册并管理 41 种专业图纸的系统指令与生成规范。
- `engines/spatial_engine.py`: **空间计算引擎**。执行 GeoJSON 处理、天际线计算和 GVI 指标挂载（已全面实现高可用缓存）。

## 🚀 启动与验证

### 1. 环境依赖安装
```bash
pip install -r requirements.txt
```

### 2. 本地启动系统
```bash
streamlit run app.py
```
*(注：需本地启动 Ollama 引擎并挂载 `gemma4:e2b-it-q4_K_M` 以获得最佳 AI 体验)*

### 3. CI/CD 测试与健康检查
```bash
# 执行代码合规性编译检查
python -m compileall app.py pages src tests tools

# 运行自动化单元测试
pytest

# 运行启动文件健康自检 (Smoke Test)
python tools/startup_smoke.py
```
