# 中共长春市委关于制定  

# 长春市国民经济和社会发展第十五个五年规划的建议  

（2025年12月5日中国共产党长春市第十四届委员会第十次全体会议通过）

---

## 一、“十五五”时期是全面建设社会主义现代化新长春的关键时期

### （1）“十四五”时期发展成效

“十四五”时期是长春在全面振兴新征程上迎难而上、加压奋进的五年，也是推动转型升级、深化改革创新的重要阶段。

主要成效包括：

- **经济发展**：GDP稳定增长，产业结构优化，“3转、4强、7新”产业体系加快构建  
- **产业升级**：汽车、装备制造转型升级，新兴产业快速发展  
- **科技创新**：建设北湖未来科学城、永春医药城等平台，全球科研城市排名提升  
- **城乡发展**：现代化都市圈建设推进，县域经济增速提升  
- **改革开放**：营商环境优化，对外开放水平提高  
- **文化发展**：历史街区活化，文化软实力提升  
- **民生改善**：就业、教育、医疗持续提升  
- **生态环境**：绿色发展推进，碳达峰试点获批  
- **安全治理**：风险防控能力增强  
- **党建工作**：全面从严治党持续深化
